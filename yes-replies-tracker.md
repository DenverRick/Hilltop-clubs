## Welcome-reply template

Personalize `[First]`, `[Club Name]`, `[Category]`, `[reply-from email]`. Keep the photo-privacy + AI-image paragraph in every reply.

```
Hi [First],

Welcome aboard! [Club Name] is now live at https://hilltopclubs.org under [Category].

[Optional one-liner addressing their specific question, e.g. "You don't
have to do anything else if you don't want to — the basic info from
Dayna's 2026 list is already on your page."]

If you'd like to flesh it out, visit:

  https://hilltopclubs.org/admin/leader

Enter the email you replied from ([reply-from email]) and you'll be able to edit:

- Long description
- Vibe / Demographics
- Banner image
- YouTube videos
- Meeting day / time / location / schedule
- TeamReach code
- External website
- Tags

A note on photos: for member privacy, we don't include photos that show actual club members. If you'd like a banner image but don't have a generic one, just let me know — I can generate a custom AI image for your club, send it to you for review, and only post it if you approve.

When a resident clicks "I'm interested" on your page, their mail app pre-fills a message to you. Your email address never appears publicly.

If anything looks off, just reply and I'll fix it.

Thanks for opting in!

Rick
```
